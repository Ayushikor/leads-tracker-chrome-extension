# Leads Tracker Chrome Extension

## Overview
Leads Tracker is a Chrome extension that helps users save and manage useful website links. Users can manually enter a URL or save the currently active browser tab with a single click.

## Features
- Save website URLs manually
- Save the current browser tab
- Open saved links in a new tab
- Delete all saved links
- Stores data using Local Storage
- Simple and responsive user interface

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Chrome Extensions API
- Local Storage API

## How to Use
1. Clone the repository.
2. Open `chrome://extensions`.
3. Enable **Developer Mode**.
4. Click **Load unpacked**.
5. Select the project folder.
6. Start saving your favorite websites!
